Об’єктно орієнтоване програмування

1) Реалізувати клас будь-якого предмета та записати деякі його властивості в якості полів. Зробити пару функцій, що будуть змінювати поля об’єктів. 2) Розробити додатковий клас чисел з математичними інтерфейсами до них.
] Примітка: для виконання завдання потрібно вивчити о методах перегрузки операторів __add__, __radd__, __sub__, __mul__, __pow__ та інш. 3) Євгенія створила клас KgToPounds з параметром kg, куди передається певна кількість кілограмів, а за допомогою методу to_pounds() вони переводяться в фунти. Щоб закрити доступ до змінної kg, вона реалізувала методи set_kg() - для завдання нового значення кілограмів, get_kg() - для виведення поточного значення кг. Через це виникла незручність: нам потрібно тепер використовувати ці 2 методи для завдання та виведення значень. Допоможіть їй переробити клас із використанням функції property() та властивостей-декораторів. Код наведено нижче. 
class KgToPounds:

    def __init__(self, kg):
        self.__kg = kg

    def to_pounds(self):
        return self.__kg * 2.205

    def set_kg(self, new_kg):
        if isinstance(new_kg, (int, float)):
            self.__kg = new_kg
        else:
            raise ValueError('Кілограми надаються тільки числами')

    def get_kg(self):
        return self.__kg
